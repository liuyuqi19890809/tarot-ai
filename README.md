# 🎴 塔罗牌占卜系统

> 78张韦特塔罗 + 12种牌阵 + 深度解读 + 精美UI

## ✨ 功能特性

### 🃏 完整塔罗牌库
- **大阿卡纳 22张**：愚者到世界，代表人生的重大课题
- **小阿卡纳 56张**：权杖、圣杯、宝剑、星币四组花色
- **正位/逆位解读**：每张牌都有正反位置的含义

### 📐 12种专业牌阵
1. **圣三角** - 最经典的三张牌阵（过去/现在/未来）
2. **四元素** - 火水土风能量分析
3. **爱情十字** - 感情问题专用
4. **凯尔特十字** - 最全面的十张牌阵
5. **是/否单张牌** - 快速决策
6. **时间流** - 趋势分析
7. **事业发展** - 工作职业规划
8. **每周运势** - 七日运程
9. **二选一决策** - 两难选择辅助
10. **关系深度分析** - 人际关系解读
11. **人生发展** - 各领域运势
12. **心灵指引** - 自我探索疗愈

### 🎨 精美UI设计
- 🌌 动态星空背景
- ✨ 渐变流光效果
- 🎴 精美的塔罗牌卡片设计
- 📱 响应式布局，支持移动端
- 🔄 流畅的抽牌动画效果

## 🚀 快速开始

### 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python -m app.main

# 访问地址
open http://localhost:9999
```

### 部署到 GitHub Pages

本项目已配置好 GitHub Actions 自动部署，只需：

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `gh-pages` 分支作为源
4. 访问 `https://your-username.github.io/tarot-ai`

## 🔧 API 接口

### 获取所有牌阵
```http
GET /api/spreads
```

### 获取所有塔罗牌
```http
GET /api/cards
```

### 抽牌占卜
```http
POST /api/draw
Content-Type: application/json

{
  "spread_type": "three_card",
  "question": "我最近的感情运势如何？"
}
```

### 健康检查
```http
GET /api/health
```
